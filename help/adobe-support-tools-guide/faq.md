# Perguntas frequentes sobre o [!DNL Adobe Business Platform] e Admin Console

+++**O que é a Identidade da Adobe e o Admin Console?**

O [!DNL Adobe Business Platform] fornece aos administradores e líderes funcionais um console de controle de última geração para gerenciar sua versão personalizada do software Workfront.

+++

+++**Por que os clientes precisam migrar para a Adobe Identity and Admin Console (AAC)?**

Os clientes experimentarão um console administrativo ainda mais seguro e intuitivo ao mudar para [!DNL Adobe Business Platform].

+++

+++**Quando os clientes precisarão mudar para o AAC?**

Os clientes serão contatados para coordenar sua mudança para a Adobe Identity e a Admin Console a partir do segundo semestre de 2022, prosseguindo até 2023.

+++

+++**O que é uma organização IMS?**

IMS é um acrônimo para Identity Management Services with Adobe Workfront que se refere a...

Uma organização é a entidade que permite a um administrador configurar grupos e usuários e controlar o logon único na Experience Cloud. A organização funciona como uma empresa para logon que abrange todos os produtos e os aplicativos da Experience Cloud. Frequentemente, a organização é o nome da empresa. No entanto, uma empresa pode ter muitas organizações.

+++

+++**Como os clientes mudarão para a AAC?**

Os clientes serão transferidos pela equipe de suporte da Adobe Workfront com assistência e treinamento em tempo real para garantir que a transição seja tranquila e que as perguntas gerais do cliente sejam respondidas.

+++

+++**Qual é o processo para mover para o AAC a partir da Arquitetura do Workfront existente?**

Os clientes receberão instruções e coordenação de uma equipe interna da Workfront designada para ajudar os clientes na mudança.

+++

+++**Que novos recursos estão atualmente disponíveis na AAC para clientes no Workfront?**

Consulte [este documento](overview.md), destacando algumas das novas adições de recursos disponíveis.

+++

+++**Quais são os recursos de treinamento disponíveis para o AAC?**

&lt;https://helpx.adobe.com/br/enterprise/using/admin-console.html >

+++

+++**Uma instância pessoal do Workfront será migrada para o AAC?**

SIM.

+++

+++**A migração para o AAC consolidará os consoles administrativos se um cliente tiver outros produtos no conjunto do Adobe?**

Sim! O console administrativo é idêntico em toda a Adobe e fornece aos administradores uma plataforma para usar o Workfront e quase todos os outros produtos da Adobe.

+++

+++**Quanto tempo leva para migrar para o AAC de um cliente típico do Workfront?**

O tempo de implementação da AAC para os clientes pode variar, mas a experiência típica terá uma experiência de teleconferência ao vivo com um cliente de cerca de duas horas.

+++

+++**Isso afetará outras tecnologias que eu integrei à solução da Workfront fora dos Produtos da Adobe?**

Os clientes que usam o aplicativo [!DNL Fusion] para se conectar a outros serviços só serão movidos quando o aplicativo [!DNL Fusion] estiver pronto para fazer essa alteração.

+++

+++**Há algum custo para os clientes mudarem para o AAC?**

Não haverá custos adicionais para os clientes ao mudar para o AAC.

+++

+++**Há algum motivo para não migrar para o AAC?**

O AAC fornecerá aos clientes as melhores atualizações e recursos aprimorados na experiência futura do Workfront e é uma ótima mudança para os clientes. Todos os clientes precisarão mudar para o AAC até o final de 2023, pois os consoles de demonstração herdados serão encerrados.
